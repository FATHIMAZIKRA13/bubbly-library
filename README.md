# 💖 Bubbly Library

A beautiful and secure Library Management System with a cute pink theme.

![Bubbly Library](https://i.imgur.com/your-image-link-here.jpg)

## ✨ Features

- **User Authentication** (Register & Login)
- **Role-based Access** (Admin & Member)
- **Book Management** (Add, View & Borrow Books)
- **Responsive Design** with charming pink theme
- **Secure** password hashing and protected routes

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **Database**: SQLAlchemy + SQLite
- **Frontend**: Bootstrap 5, Custom CSS
- **Authentication**: Flask-Login
- **Deployment**: Render.com
